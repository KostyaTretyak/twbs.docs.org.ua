Bootstrap є найбільш популярним HTML, CSS, та JS фреймворком для зовнішнього вигляду та інтерактивності веб‑сторінок.

Скачати Bootstrap

Поточна v3.3.2

Спроектований для будь-кого та для будь-якого пристрою.

Bootstrap допомагає швидше та простіше розробляти зовнішній вигляд веб-сторінок. Він підходить для людей з будь-яким рівнем досвіду, для пристроїв будь-яких форматів, та проектів будь-якого розміру.

Препроцесори

Bootstrap йде з чистим CSS, але його сирцевий код використовує два найбільш популярні CSS‑препроцесори Less та Sass. Можна швидко почати з готовими CSS, або ж віддати перевагу побудові стилів із сирців.

Один фреймворк, будь-який пристрій.

Bootstrap легко та ефективно масштабується на вашому сайті на основі єдиного базового коду: від телефонів і планшетів до настільних комп'ютерів – все завдяки CSS медіа-запитам.

Повен можливостей

Разом з Bootstrap ви отримаєте докладну та гарну документацію для загальних елементів HTML, дюжину піддатливих компонентів HTML та CSS, і афігенні плагіни jQuery =).

Bootstrap має відкритий код, який розміщується, розробляється, та обслуговується на GitHub.
